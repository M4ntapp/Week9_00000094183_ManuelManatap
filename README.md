## Cara Menjalankan

### 1. Install dependencies
```bash
npm install
```
> Jika ada error peer deps, gunakan:
> ```bash
> npm install --legacy-peer-deps
> ```

### 2. Jalankan app
```bash
npx expo start
```

### 3. Buka di device/emulator
- Scan QR code dengan **Expo Go** (Android/iOS)
- Tekan `a` untuk membuka di Android emulator
- Tekan `i` untuk membuka di iOS simulator
